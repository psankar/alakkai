alakkai (அளக்கை)


* alakkai is a free and open source project hosted at: https://github.com/psankar/alakkai

* A tool to create surveys, let people vote and see the results. The name alakkai is the Tamil term (அளக்கை) for survey

Programming

* The program is written using the go programming language for the server-side software development. Javascript and HTML are used for the web front end.
* mongodb is used for storing the surveys and the responses, serving as the backend.
